This guide walks you through setting up and running a basic simulation with an agent and RGB-D sensors.

## Prerequisites

Make sure you have [installed Habitat-Sim](/installation) before proceeding.

## Download test data

Download the test scenes and example objects using the data download utility:

```bash
# Download test 3D scenes
python -m habitat_sim.utils.datasets_download \
  --uids habitat_test_scenes \
  --data-path data/

# Download example objects
python -m habitat_sim.utils.datasets_download \
  --uids habitat_example_objects \
  --data-path data/
```

## Basic simulation

The following example creates a simulator with an RGB camera and depth sensor, places an agent in a scene, and steps through a sequence of actions.

```python
import habitat_sim
import numpy as np

# 1. Configure the simulator backend
sim_cfg = habitat_sim.SimulatorConfiguration()
sim_cfg.scene_id = "data/scene_datasets/habitat-test-scenes/skokloster-castle.glb"
sim_cfg.enable_physics = False

# 2. Define sensors
rgb_sensor_spec = habitat_sim.CameraSensorSpec()
rgb_sensor_spec.uuid = "color_sensor"
rgb_sensor_spec.sensor_type = habitat_sim.SensorType.COLOR
rgb_sensor_spec.resolution = [480, 640]
rgb_sensor_spec.position = [0.0, 1.5, 0.0]  # eye height

depth_sensor_spec = habitat_sim.CameraSensorSpec()
depth_sensor_spec.uuid = "depth_sensor"
depth_sensor_spec.sensor_type = habitat_sim.SensorType.DEPTH
depth_sensor_spec.resolution = [480, 640]
depth_sensor_spec.position = [0.0, 1.5, 0.0]

# 3. Configure the agent
agent_cfg = habitat_sim.AgentConfiguration()
agent_cfg.sensor_specifications = [rgb_sensor_spec, depth_sensor_spec]
agent_cfg.action_space = {
    "move_forward": habitat_sim.ActionSpec(
        "move_forward", habitat_sim.ActuationSpec(amount=0.25)
    ),
    "turn_left": habitat_sim.ActionSpec(
        "turn_left", habitat_sim.ActuationSpec(amount=10.0)
    ),
    "turn_right": habitat_sim.ActionSpec(
        "turn_right", habitat_sim.ActuationSpec(amount=10.0)
    ),
}

# 4. Create the simulator
cfg = habitat_sim.Configuration(sim_cfg, [agent_cfg])
sim = habitat_sim.Simulator(cfg)

# 5. Get the agent and set its initial state
agent = sim.initialize_agent(0)
agent_state = habitat_sim.AgentState()
agent_state.position = np.array([-0.6, 0.0, 0.0])  # world coordinates
agent.set_state(agent_state)

# 6. Step the simulation
observations = sim.step("move_forward")
print("Color observation shape:", observations["color_sensor"].shape)
print("Depth observation shape:", observations["depth_sensor"].shape)

sim.close()
```

## Interactive viewer

Habitat-Sim includes an interactive viewer for exploring scenes:

<Tabs>
  <Tab title="Python viewer">
    ```bash
    python examples/viewer.py --scene data/scene_datasets/habitat-test-scenes/skokloster-castle.glb
    ```
  </Tab>
  <Tab title="C++ viewer (after pip install)">
    ```bash
    viewer data/scene_datasets/habitat-test-scenes/skokloster-castle.glb
    ```
  </Tab>
</Tabs>

**Controls:**
- `W` / `A` / `S` / `D` — Move forward / left / backward / right
- Arrow keys or left mouse click — Look up / down / left / right
