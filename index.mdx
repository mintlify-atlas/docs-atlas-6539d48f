## What is Habitat-Sim?

Habitat-Sim is a simulator designed for training and benchmarking embodied AI agents. It achieves several thousand frames per second (FPS) single-threaded and over 10,000 FPS multi-process on a single GPU when rendering from the Matterport3D dataset.

Key capabilities:

- **3D scene support** — Built-in support for HM3D, HSSD, Matterport3D, Gibson, Replica, and other datasets
- **Configurable sensors** — RGB, depth, semantic, audio, fisheye, and equirectangular sensors
- **Physics simulation** — Rigid-body and articulated object mechanics via Bullet Physics
- **Robot support** — URDF-based robots including mobile manipulators, fixed-base arms, and quadrupeds
- **NavMesh navigation** — Collision-aware pathfinding with geodesic distance computation
- **Python API** — Full Python bindings backed by a high-performance C++ rendering engine

## Performance

Habitat-Sim is designed for speed. On a Matterport3D scene with a single GPU:

| Configuration | 128px | 256px | 512px |
|---|---|---|---|
| RGB (1 proc) | 4,093 FPS | 1,987 FPS | 848 FPS |
| RGB (3 procs) | 10,638 FPS | 3,428 FPS | 2,068 FPS |
| RGB + Depth (1 proc) | 2,050 FPS | 1,042 FPS | 423 FPS |
| RGB + Depth + Semantics (1 proc) | 709 FPS | 596 FPS | 394 FPS |

## Ecosystem

Habitat-Sim is typically used alongside [Habitat-Lab](https://github.com/facebookresearch/habitat-lab), a high-level library for end-to-end embodied AI experiments including task definition, agent training, and evaluation.
